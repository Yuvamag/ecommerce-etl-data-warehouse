# 🏬 E-Commerce End-to-End ETL Data Warehouse Project

## 📌 Project Overview
This project demonstrates a complete end-to-end ETL pipeline for an E-Commerce dataset using SQL Server and Python.  
The solution implements a Star Schema Data Warehouse with SCD Type 2 handling and Power BI dashboard reporting.

---

## 🏗️ Architecture

- Source Data (Raw CSV)
- Python Data Profiling & Cleaning
- SQL Server Staging Layer
- Dimension & Fact Tables (Star Schema)
- SCD Type 2 Implementation
- Analytical Views
- Power BI Dashboard

---

## 🛠️ Tech Stack

- Python (Pandas, Data Profiling)
- SQL Server Management Studio 21
- T-SQL (Stored Procedures, Views, Constraints)
- Power BI
- GitHub

---
## 📂 Project Structure
```
python/     -> Data profiling & cleaning scripts  
sql/        -> Schema, dimensions, fact, staging, validations, views  
powerbi/    -> Power BI dashboard file (.pbix)
```
